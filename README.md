[+] bIRC - Easy IRC Client for SSL IRC server
[+] Usage: ./bIRC.rb <host> <port> <nick>


All irc's commands...
/privmsg #chan <msg>
/mode #chan +o <nick>

Commands:
/c #channel	Set channel for chatting

Brevius commands:
/j <chan>	join
/p <chan>	part
/q		quit
